# Guard1 — Multiple Perspectives on Political Issues

A news platform exploring political and social issues through multiple perspectives.

## Project Structure

```
guard1/
├── public/                    # Web root (upload to hosting)
│   ├── index.html            # Homepage
│   ├── css/styles.css        # Purple/Grey wireframe design
│   ├── js/script.js          # Interactive features
│   └── articles/             # Article pages
├── reference/                 # Editorial reference
├── politicians/               # Politician profiles
├── templates/                 # Content templates
└── README.md
```

## Design

- **Purple tones** - Calming, empowering
- **Grey** - Neutral, authoritative  
- **Wireframe aesthetic** - Clean lines, minimal

## Getting Started

Open `public/index.html` in your browser to view the site.

## Deployment to InfinityFree

1. Log in to InfinityFree Control Panel
2. Go to Files → htdocs
3. Upload all files from `public/` folder
4. Site live at `youraccount.infinityfreeapp.com`

## Creating Articles

1. Copy `templates/article-template.html` to `public/articles/[topic].html`
2. Edit content with Facts, Perspectives, Nuance, Reflection sections

## Content Guidelines

1. **The Facts** - Verifiable, neutral information
2. **Perspectives** - Different viewpoints with context
3. **The Nuance** - Tensions and questions missed
4. **Reflection** - Questions for readers
